# Task Manager

Task Manager is a simple, interactive task tracking web app built with HTML, CSS, and JavaScript. It helps you organize your daily work, manage priorities, and keep track of progress in one place.

## Features

- Add tasks with priority and due date
- Edit task names inline
- Mark tasks complete by clicking the task row or checkbox
- Delete tasks and clear completed items
- Search and filter tasks
- Sort tasks by newest, oldest, priority, or due date
- View task summary cards and quick widgets
- Toggle dark mode for a different visual theme
- Save tasks automatically in local storage

## Project Structure

- `index.html` - Main app layout and UI sections
- `style.css` - All visual styling and responsive behavior
- `script.js` - Task logic, filters, sorting, widgets, and storage

## How to Use

1. Open `index.html` in a browser.
2. Add a task using the form at the top.
3. Click a task row to mark it complete.
4. Use the Edit button to change the task name inline.
5. Use the cards, widgets, search, and filter controls to organize your list.

## Notes

- Tasks are stored in the browser using local storage.
- Refreshing the page will keep your saved tasks.
- This is a frontend-only project, so no backend or login system is connected yet.
